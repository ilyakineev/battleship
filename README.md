# Морской Бой (Battleship)

Простая консольная игра "Морской Бой", написанная на языке Rust.

## Игровой процесс

Это классическая игра "Морской Бой". Игра автоматически размещает корабли для вас и для компьютера на сетке 10x10. Вы и ИИ по очереди называете координаты, чтобы найти и потопить корабли друг друга.

Чтобы сделать выстрел, введите координаты в формате `<Буква><Номер>`, например: `A5`, `D10` или `G7`. Игрок, первым потопивший все корабли противника, побеждает.

## Необходимые программы

Для компиляции и запуска этого проекта у вас должен быть установлен инструментарий Rust. Самый простой способ получить его — использовать официальный установщик `rustup`.

- **Для macOS, Linux или других Unix-подобных ОС:**
  ```sh
  curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
  ```

- **Для Windows:**
  Посетите сайт [https://rustup.rs/](https://rustup.rs/) и следуйте инструкциям для загрузки и запуска `rustup-init.exe`.

## Как скомпилировать

После установки Rust вы можете скомпилировать игру с помощью Cargo — инструмента сборки и менеджера пакетов Rust.

1.  Клонируйте репозиторий на свой локальный компьютер.
2.  Откройте терминал и перейдите в каталог проекта.
3.  Выполните команду сборки:
    ```sh
    # Для отладочной сборки
    cargo build
    ```
    Для сборки, оптимизированной для производительности, используйте флаг `--release`:
    ```sh
    # Для релизной (оптимизированной) сборки
    cargo build --release
    ```
    Cargo сам загрузит зависимости и скомпилирует исходный код.

## Как запустить

Есть два способа запустить игру.

#### Способ 1: С помощью Cargo (Рекомендуется)

Эта команда компилирует и запускает игру за один шаг. Это самый простой метод.
```sh
cargo run
```
Если вы хотите запустить оптимизированную версию:
```sh
cargo run --release
```

#### Способ 2: Запуск исполняемого файла напрямую

После сборки проекта исполняемый файл помещается в каталог `target`.

- **Для macOS или Linux:**
  ```sh
  # Для отладочной сборки
  ./target/debug/battleship

  # Для релизной сборки
  ./target/release/battleship
  ```

- **Для Windows:**
  ```sh
  # Для отладочной сборки
  .\\target\\debug\\battleship.exe

  # Для релизной сборки
  .\\target\\release\\battleship.exe
  ```
